# QuestAR Workshop Materials
**Build & Host Your Own AR Treasure Hunt in One Day**

Welcome to the QuestAR workshop! This folder contains all the materials, starter code, and guides needed to run or participate in this 4-session course designed for absolute beginners.

- [**participant-setup.md**](participant-setup.md): **Start Here!** Pre-workshop checklist for all participants.

## 🚀 Quick Start
1. **Prerequisites**: Ensure you have [GitHub](https://github.com) and [Vercel](https://vercel.com) accounts.
2. **Session 1**: Build your portfolio in `session1/index.html` and follow the [Vercel guide](session1/hosting-vercel.md).
3. **Session 2**: Start building the game interface in `session2/index.html`.
4. **Session 3**: Finalize the AR experience in `session3/ar-game.html`.
5. **Session 4**: Remix the code and launch your own idea following the [activity guide](session4/activity.md).

## 🛠️ Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **3D Engine**: [Three.js](https://threejs.org/)
- **AR Library**: [Mind.AR](https://hiukim.github.io/mind-ar-js-doc/)
- **Hosting**: Vercel & GitHub Pages
