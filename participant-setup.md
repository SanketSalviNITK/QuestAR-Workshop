# Participant Setup Checklist 🚀
**Please complete these steps before the workshop begins!**

---

## 1. 🛠️ VS Code Extensions
Open the **Extensions** view in VS Code (click the square icon on the left or press `Ctrl+Shift+X`) and install the following:

1.  **Live Server** (by Ritwick Dey): To see your code changes in the browser instantly.
2.  **Prettier - Code Formatter**: To keep your code clean and readable.
3.  **Auto Rename Tag**: To save time and prevent HTML errors.

---

## 2. 🌐 Account Setup
You will need active accounts for the following services (please link them if possible):

1.  **[GitHub](https://github.com)**: For storing your code.
2.  **[Vercel](https://vercel.com)**: For hosting your websites. Sign up with your GitHub account for a seamless experience.

---

## 3. 🖥️ Terminal Configuration
Open your terminal in VS Code (`Ctrl + `) and run these commands to set up your Git identity:

```bash
# 1. Verify Git is installed
git --version

# 2. Configure your name (Replace with your actual name)
git config --global user.name "Your Name"

# 3. Configure your email (Replace with your actual email)
git config --global user.email "your-email@example.com"

# 4. Check your settings
git config --list
```

---

## 4. 📱 Hardware Preparation
*   **Smartphone**: Bring your phone and ensure you have a QR code scanner (most camera apps have this built-in).
*   **Browser**: Please use **Google Chrome** or **Microsoft Edge** for the best compatibility with Augmented Reality features.
