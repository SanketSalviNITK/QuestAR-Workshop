# Teaching Notes: Session 1 - Your Digital Identity 🚀
**Duration**: 90 Minutes

## 1. Introduction & The "3 Pillars" (10 Mins)
*   **The Hook**: "Today, you aren't just building a website; you're building a digital home and an Augmented Reality adventure. But before we fly, we have to walk."
*   **The Concept**: Explain that every website is made of three things:
    1.  **HTML (The Skeleton)**: Headings, paragraphs, images.
    2.  **CSS (The Skin)**: Colors, fonts, spacing, animations.
    3.  **JavaScript (The Brain)**: Clicks, logic, and AR magic.
*   **Teaching Tip**: Use the analogy of a house. HTML is the bricks and beams; CSS is the paint and furniture; JS is the light switches and smart home system.

## 2. HTML: Building the Skeleton (20 Mins)
*   **Focus**: Semantic HTML.
*   **Key Concept**: Every tag has a meaning. Don't just use `<div>` for everything.
*   **Live Demo Points**:
    *   Show how `<h1>` creates a hierarchy.
    *   Explain `<nav>` for the navigation bar.
    *   Explain `<section>` for logical breaks.
*   **Speaker Note**: "HTML is about structure. If you get the structure right, the styling becomes 10x easier."

## 3. CSS: Premium Aesthetics (20 Mins)
*   **Focus**: The "Wow" factor.
*   **Key Concepts**:
    *   **Google Fonts**: Moving away from the browser's default "boring" font.
    *   **The Box Model**: Padding vs Margin (Use the 'package' analogy: padding is the bubble wrap inside, margin is the space between boxes).
    *   **Colors**: Introduce HSL or Hex codes. Encourage them to pick a "Primary" color.
*   **The "Glassmorphism" Secret**: Mention `backdrop-filter: blur(10px)`—this is what gives modern apps (like Apple or Windows 11) their premium look.

## 4. GitHub: Your Developer Resume (15 Mins)
*   **Focus**: Fearless coding.
*   **The "Time Machine" Analogy**: Explain that Git allows you to "Save" your progress so if you break something, you can always go back.
*   **The Workflow**:
    1.  `git init` (Start the engine).
    2.  `git add .` (Pack the bags).
    3.  `git commit` (Take a snapshot).
    4.  `git push` (Upload to the cloud).
*   **Speaker Note**: "Your GitHub profile is your digital CV. Every commit you make today is a proof of your growth."

## 5. Vercel: The Stage (15 Mins)
*   **Focus**: Instant gratification.
*   **Key Concept**: "Continuous Deployment."
*   **The Workflow**:
    *   Import the repo from GitHub.
    *   Hit 'Deploy'.
    *   Show them the live link on their phones.
*   **Teaching Tip**: Ask students to share their live links in the class group chat/Discord immediately. This builds excitement.

## 6. Wrap-up & Troubleshooting (10 Mins)
*   **Common Bugs**:
    *   Missing closing tags in HTML.
    *   Spelling errors in CSS properties.
    *   Path errors (linking `style.css` incorrectly).
*   **Homework**: Encourage them to add one personal thing (a bio or an image) to their portfolio before Session 2.
