# QuestAR: From Portfolio to Augmented Reality
**A One-Day 4-Session Workshop for Beginners**

---

## 📋 Course Description
Welcome to the world of web development! This workshop is designed for absolute beginners. We will start by building your personal digital presence (a portfolio), then dive into the cutting-edge world of Augmented Reality (AR). By the end of the day, you will be hosting your own creative AR ideas for the world to see.

## 🕙 Schedule

### Session 1: Your Digital Identity
*09:00 AM - 10:30 AM*
- **Objective**: Build and host your first professional website.
- **Topics**:
  - **HTML5**: The structure of the web (Text, Links, Images).
  - **CSS3**: Making it look premium (Fonts, Flexbox, Colors).
  - **Vercel & GitHub**: Connecting your code to the cloud for instant hosting.
- **Outcome**: A live, personal portfolio website hosted on Vercel.

### Session 2: Entering the 3D World
*11:00 AM - 12:30 PM*
- **Objective**: Build the interface and 3D assets for your AR game.
- **Topics**:
  - **UI Design**: Creating a "Glassmorphism" game dashboard.
  - **Three.js**: Introduction to 3D space, geometries (The Treasure), and lighting.
  - **JS Logic**: Basic event handling and state management.
- **Outcome**: A functional 3D dashboard ready for AR integration.

### Session 3: Bridging Reality (AR Integration)
*01:30 PM - 03:30 PM*
- **Objective**: Bring your 3D treasure into the real world.
- **Topics**:
  - **Mind.AR**: Web-based image tracking fundamentals.
  - **AR Logic**: Mapping your 3D gem to physical markers.
  - **Collection Mechanics**: Building the "Score" system.
- **Outcome**: A fully working AR Treasure Hunt prototype.

### Session 4: The Creator's Lab (Open Activity)
*04:00 PM - 05:00 PM*
- **Objective**: Remix, build, and host your own unique idea.
- **Activity**:
  - **Group Work**: Brainstorm a new use for AR (Art, Education, Games).
  - **Implementation**: Modify the codebase to use new 3D models or markers.
  - **Global Launch**: Host the new project on Vercel/GitHub.
- **Outcome**: A unique, student-led AR project live on the web.

## 🛠️ Prerequisites
- A laptop with VS Code installed.
- A GitHub account and a Vercel account.
- A smartphone for testing AR.

---

## 📦 Setup Instructions
1. Create a [GitHub Account](https://github.com).
2. Create a [Vercel Account](https://vercel.com) (Link it to your GitHub).
3. Install the **Live Server** extension in VS Code.
